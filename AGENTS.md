# Developer Instructions

## Responsive Baseline

- Treat `320px` viewport width as a required layout target for all pages and components.
- No horizontal overflow is allowed at `320px` (content must remain fully visible without clipping).
- CTA/button labels must remain readable at `320px` (allow wrapping when needed; do not rely on fixed minimum widths).
- When changing layout, spacing, typography, or components, verify rendering at `320px`, `375px`, and desktop before merging.
